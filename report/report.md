# Week 4 Data Visualization Project
## Sales Analysis Report

### 📌 Project Overview
This project analyzes sales data using Python.  
It demonstrates a complete workflow: loading, cleaning, analyzing, visualizing, and writing insights.

---

### 📂 Data Cleaning
- Missing values removed using `dropna()`.
- Date column converted to datetime format.
- Ensured dataset consistency before analysis.

---

### 📊 Analysis
- **Sales by Product** → Total sales calculated per product category.  
- **Monthly Sales Trend** → Aggregated sales by month to identify seasonal patterns.  
- **Product Share** → Distribution of sales across categories.

---

### 📈 Visualizations
- Bar chart → Sales by product (`visualizations/bar_chart.png`)  
- Line chart → Monthly sales trend (`visualizations/line_chart.png`)  
- Pie chart → Product share (`visualizations/pie_chart.png`)

---

### 📝 Insights
- Product X generated the highest revenue.  
- Sales peaked in March, showing seasonal demand.  
- Category Y contributed the least, potential area for improvement.

---

### ✅ Conclusion
This project demonstrates how Python (Pandas + Matplotlib) can be used for end‑to‑end data analysis.  
The workflow is professional, reproducible, and recruiter‑friendly.
